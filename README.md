# Eventhub

## Avvio rapido

### Backend

1. Posizionati nella cartella del backend:
   ```bash
   cd backend
   ```
2. Installa le dipendenze Python:
   ```bash
   python3 -m pip install -r requirements.txt
   ```
3. Avvia l'app Flask:
   ```bash
   python3 run.py
   ```

L'applicazione sarà disponibile su `http://127.0.0.1:5000`.

### Frontend

1. Posizionati nella cartella frontend:
   ```bash
   cd eventhub-frontend
   ```
2. Installa le dipendenze Node:
   ```bash
   npm install
   ```
3. Avvia l'app Angular:
   ```bash
   npm start
   ```

L'app front-end sarà disponibile su `http://localhost:4200` o sulla porta assegnata dal server di sviluppo.
